# @slop-lab/dim-cli

The `dim` command-line interface for isolated, persistent AI development
workspaces.

```bash
npm install --global "@slop-lab/dim-cli@0.1.0"
dim --help
```

Use an exact version that your project has reviewed; do not track `latest`.
See the mandatory [adoption and trust requirements](https://github.com/slop-lab/dev-infra-manager/blob/main/docs/adoption.md).

Documentation and source are available in the
[project repository](https://github.com/slop-lab/dev-infra-manager).
